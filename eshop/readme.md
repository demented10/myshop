


## Слои

### Domain
В доменном слое описываются модели и бизнес правила.
Модели анемичные.
### Infrastructure
В инфрастуктурном слое происходит доступ к данным и реализуются внешние зависимости
Используется Entity Framework для работы с БД
### Application
В слое приложение реализуются сервисы и координируется работа между другими слоями
